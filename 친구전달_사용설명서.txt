   친구 전달 패키지
===============================================

안녕하세요, 박상철 원장님!

요청하신 환자 주소 분석 도구를 완성했습니다.

===============================================

📦 전달 내용
---------------

1. **웹 앱 URL** (추천)
   → [배포 후 URL 입력 예정]
   → 설치 없이 브라우저에서 바로 사용

2. **백업 ZIP 파일**
   → address-geo-analyzer.zip
   → 인터넷 없이 사용 시 (로컬 설치)

===============================================

🌐 웹 앱 사용법 (가장 쉬움)
---------------

1. 위 URL 클릭
2. "파일 선택" 버튼 → 환자 주소 엑셀/CSV 업로드
3. "🚀 분석 시작" 버튼 클릭
4. 결과 확인 후 "📥 결과 다운로드" 클릭

⚠️ 필수: 엑셀 첫 줄에 "주소" 컬럼 필요
(예시: 경기도 화성시 동탄구 동탄신리천로8길 15)

===============================================

📊 결과물
---------------

✅ 동별 환자 분포
   - 신동, 목동 등 각 동의 환자 수
   - 각 동의 아파트 목록 (환자 수 포함)

✅ 아파트별 환자 분포
   - 아파트명 + 소재 동 + 환자 수
   - 상위 20개 단지

✅ 차트 시각화
   - 막대 그래프로 한눈에 확인

✅ CSV 파일 다운로드
   - 엑셀에서 추가 분석 가능

===============================================

💡 활용 예시
---------------

【 마케팅 전략 】
- 환자 많은 동 → 전단지 배포
- 환자 많은 아파트 → 입주민 이벤트

【 진료 시간 조정 】
- 특정 동 환자 집중 → 셔틀 버스 검토

【 경쟁 분석 】
- 환자 적은 동 → 경쟁 병원 확인

===============================================

❓ 문제 해결
---------------

【 웹 앱이 느려요 】
→ 정상입니다. 100건 기준 30초~1분 소요
   (무료 API 호출 제한 때문)

【 일부 주소가 분석 안 돼요 】
→ 신축 건물이거나 도로명 주소 오타
→ 수동 확인 필요

【 웹 앱이 안 열려요 】
→ 7일간 미접속 시 자동 sleep
→ URL 다시 클릭하면 30초 후 재시작
